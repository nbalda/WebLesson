WEB

web нь Hyper Text Markup Language гэсэн үг бөгөөд дэлхийн өргөн документууд үүсгэхийн тулд тэмдэглэгээний хэл юм. HTML нь хэд хэдэн тэмдэглэгээнүүдээс бүрддэг бөгөөд та тэдгээрийг өөрийн сонгосон документ үүсгэхийн тулд ашиглаж болно. Интернет баримт нь Hyper Text-ийн зарчмыг ашиглан төрөл бүрийн баримт бичгүүдийг холбож, янз бүрийн мэдээллийг эрж хайж, харуулдаг. Эдгээр олон янзын илэрхийллийн хэрэглээ, боломжууд нь биднийг World Wide Web-ийг богино хугацаанд түгээх боломжийг олгосон.
